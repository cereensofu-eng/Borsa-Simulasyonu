# Borsa Simülasyonu 
Python ile geliştirilmiş, nesne yönelimli proglamlama (OOP) mantığıyla hazırlanmış konsol tabanlı bir borsa simülasyonu.

## Proje Hakkında
Bu proje kullanıcıların sanal bir borsa ortamında hisse senedi alıp satmasını sağlayan bir simülasyondur.Hisse fiytları rastgele güncellenebilir ve kullanıcı portföyü gerçek zamanlı olarak takip edebilir.

## Özellikler 
-Hisse senetlerini inceleme 
-Hisse arama
-Hisse satın alma 
-Hisse satma 
-Portföy görüntüleme 
-Kullanıcı bakiyesi
-Rastgele fiyat güncelleme 
-Modüler python yapısı

## Kullanılan Teknolojiler
